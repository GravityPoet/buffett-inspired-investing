## 🌟 核心增益：它能为你带来什么？

1. **🔒 终结幻觉，强制只读「第一手信源」**：
   自带高确定性 Python 脚手架，一秒直达 美股 (SEC)、港股 (HKEX) 与 A股 (沪/深) 官方财报入口，让大模型只基于 10-K、Proxy 和电话会议录音进行思考。
2. **🧠 破除财务指标迷信，穿透「商业本质」**：
   剥夺 AI 谈论 PE/PB 估值的权力，强制其围绕 **护城河持久度 (Moat)**、**管理层坦诚度 (Candor)** 与 **资本配置能力 (Capital Allocation)** 展开定性分析。
3. **⚔️ 真正的护城河「行业级 Playbook」**：
   告别通用模版。分析 SaaS 时强制审视“转换成本”，分析保险时深挖“浮存金成本”，让 AI 掌握真正的行业认知门槛。
4. **🛡️ 培养极高信噪比的「Pass 文化」**：
   赋予模型快速做出决断的底气。不为了分析而分析，一秒钟筛除平庸生意，只对**进入击球区极佳的生意**出击。

## 💎 立竿见影的效果 (Before & After)

没有对比就没有伤害，以下是接入本框架前后的巨大差异：

**❌ 当你使用普通 AI 提示词时：**
> *“帮我分析一下苹果公司的股票还能不能买。”*
> **AI 回答**：“苹果(AAPL)目前股价 180 美元，受 iPhone 销量影响，近期 K 线呈上涨趋势，华尔街给出了强力买入建议……”（全是二手废话与赌场思维）

**✅ 当你接入本仓库工作流后：**
> *(大模型将深度研读 10-K、回购数据，并融合顶级商业认知后，产出纪律严明的深度备忘录)*
> 
> **🍎 案例一：Apple Inc. (AAPL)**
> **🎯 最终结论: 放入观察池 / 暂不击球 (WATCHLIST)**
> 
> - 🟢 **生意的护城河 (极强的消费者特许权)**：这并非一家科技企业，而是一家拥有难以置信的消费者心智占有率的消费品公司。iOS 生态系统创造了极高的转换成本。就像查理说的，当极高的品牌黏性遇上无形的数字生态，它产生了一种极其持久的“经济特许权 (Economic Franchise)”。
> - 🟢 **管理层资本配置 (极度自律)**：库克团队深知当这头大象无法把所有现金以高回报率再投资回主业时，最聪明的做法是什么。过去数年间，他们极其无情且自律地回购自家股票，没有拿股东的钱去搞价值毁灭的盲目并购。所有者权益被实质性放大了。
> - 🔴 **估值的安全边际 (无容错率)**：在当前高达 30 倍以上市盈率的价格，市场已经提前透支了它未来数年的完美增长。一笔好生意在糟糕的价格下买入，依然可能是一笔平庸的投资。不要因为害怕错过而挥棒。
> 
> **📦 案例二：Amazon (AMZN)**
> **🎯 最终结论: 放入观察池 / 寻找结构性错杀机会**
> 
> - 🟢 **生意的护城河 (规模经济共享)**：贝佐斯打造了类似 GEICO 或 Nebraska Furniture Mart 般的经典护城河——“规模经济共享 (Shared Economies of Scale)”。更恐怖的是，AWS 则是一门几乎不可替代的数字基础设施生意（如同现代数字公用事业），转换成本高得惊人。
> - 🟡 **阅读财报的壁垒 (会计利润的迷惑性)**：绝大多数华尔街分析师受限于通用会计准则 (GAAP)，对着短期净利润的波动大加鞑伐。而以我们的所有者视角看，管理层为了强化护城河，长年将极其丰厚的所有者盈余（Owner Earnings）立刻化为维持未来垄断地位的资本性支出。
> - 🔴 **能力圈与估值 (过分昂贵的确定性)**：在目前的市值下，你正在为未来的垄断确定性支付极高昂的价格。除非市场情绪极度悲观给出明显的买点，否则这是一门好得无需证实，但也贵得下不了手的生意。
>
> **🎮 案例三：NVIDIA (NVDA)**
> **🎯 最终结论: 放入“太难 (Too Hard)”纸篓**
> 
> - 🔴 **打破“能力圈 (Circle of Competence)”原则**：这正是查理最经常扔进“太难”纸篓的典型案例。我们无法在未来十年的跨度内，以极高的确定性预测底层 AI 资本支出的景气周期，以及软件生态在未来算力突围下的抗脆弱性。
> - 🔴 **避免预测周期顶端**：不管当前的营收增速多么亮眼，这种“卖铲子”的盈利狂潮高度依赖下游少数几个科技巨头的“军备竞赛”。当我们无法预测下一个大规模削减设备投资的拐点何时到来时，为其支付数十倍的市销率 (P/S) 违背了基本的安全边际法则。
> - 💡 **下一步行动**：不必在这上面浪费时间。投资不需要你在每一个高难度击球上得分，跳过不需要付出任何代价。去寻找那些你能看懂其未来十年自由现金流的“一英尺跨栏”。

## 🚀 极速启动 (Quick Start)

### 1. 安装为本地技能库
将本仓库克隆至你的 Agent 工作流目录：
```bash
# 典型的本地路径
$CODEX_HOME/skills/buffett-inspired-investing
```

### 2. 跨市场公司研报引导脚手架
使用极具确定性的 python 引导脚本，直接获取官方信源（美股/港股/A股）的阅读切入点，避免依赖大模型不稳定的网页搜索：

```bash
python3 scripts/official_market_bootstrap.py AAPL
python3 scripts/official_market_bootstrap.py 0700.HK
python3 scripts/official_market_bootstrap.py 600519.SH
python3 scripts/official_market_bootstrap.py 300750.SZ --json
```

> [!NOTE]
> **SEC 助手注意事项:** 在抓取 SEC 财报快照前请设置合规的 User Agent：
> `export SEC_USER_AGENT="Your Name research@example.com"`  
> `python3 scripts/sec_company_snapshot.py AAPL`

### 3. 给 AI 下达第一个指令
你可以在 [examples/example-prompts.md](examples/example-prompts.md) 中找到完整的提问灵感。推荐用以下方式开局：

## 🧠 刻意的架构设计抉择

1. **巴菲特启发式，而非巴菲特角色扮演 (Cosplay)**。使用他的框架可以保持输出的诚实性，强迫 AI 说“你好，我是巴菲特”对分析深度毫无帮助。
2. **第一手来源优先**。此工作流强制模型走这条路线：`10-K/20-F -> 业绩电话会文字录 -> 投资者关系材料 -> 监管文件 -> 然后才是二手分析`。
3. **“放弃 (Pass)” 是一种非常优质的成功产出**。我们认为输出 `强力匹配`、`加入观察池`、`放弃`、`超出能力范围` 都是同样合理且成功的结论。
4. **拒绝虚假精确度**。我们允许给出估值区间范围或情境推演，但明令禁止 AI 编造精确的未来 12 个月目标价。
