# 中文参考说明

这一层不是随手翻译，而是给中文研究流直接使用的核心参考面板。

如果你希望整份输出都用中文，优先从这里开始。

## 建议阅读顺序

1. `framework.md`
2. `memo-template.md`
3. `source-map.md`
4. `official-principles.md`
5. `industry-playbooks.md`
6. `mental-models.md`

## 什么时候用这一层

- 用户直接用中文提问
- 你准备输出中文研究备忘录
- 你需要把 Buffett 风格框架讲得更接近中文投资者的阅读习惯
- 你在分析港股或 A 股时，希望官方来源路径更直观

## 和英文层的关系

- 英文层仍然是主框架原稿
- 中文层是便于执行的镜像版本
- 遇到术语模糊时，以英文层的原始结构为准，再用中文层解释

## 搭配使用建议

- 先用 `python3 scripts/official_market_bootstrap.py <代码>` 确认官方入口
- 再按这里的 `source-map.md` 去读原始披露
- 最后按 `memo-template.md` 输出完整中文备忘录
